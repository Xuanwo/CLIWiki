判断一个变量是否为空

empty ( mixed $var )

需要注意的是以下值会被判定为空值：

  • "" (an empty string)
  • 0 (0 as an integer)
  • 0.0 (0 as a float)
  • "0" (0 as a string)
  • NULL
  • FALSE
  • array() (an empty array)
  • $var; (a variable declared, but without a value)

将两个对象合并在一起

foreach($objB as $var=>$value){
    $objA->$var = $value;
}
